# little-byte-demo

一些 v8 字节码的相关 Demo

测试环境为 node v12

Compile `demo/hello-export.js`

```bash
npm run compile
```

Run bytecode

```bash
npm run demo
```

Run Benchmark

```bash
npm run benchmark
```
